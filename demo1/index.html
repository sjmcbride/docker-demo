🚀 Demo1 Website

Welcome to Demo1.smclab.net

This is the first demo website in our Docker stack. It's running behind Traefik with automatic SSL certificates and demonstrates modern containerized web application architecture.

🎯 Core Features:

🔒 Automatic SSL

Let's Encrypt certificates with auto-renewal

🚀 Container Ready

Optimized Docker containers with Alpine Linux

🌐 Load Balanced

Traefik reverse proxy with service discovery

📊 Database Shared

PostgreSQL connection across all services

⚙️ Technical Stack

Frontend: HTML5, CSS3, JavaScript (Vanilla)
Web Server: Nginx (Alpine Linux)
Proxy: Traefik v3.0 with automatic service discovery
Database: PostgreSQL 15 with persistent storage
SSL: Let's Encrypt automatic certificate management
Orchestration: Docker Compose with multi-network architecture

📊 Database Connection

Connected to shared PostgreSQL database: demo_db

Database contains sample data for all three demo sites with visitor tracking and analytics.
